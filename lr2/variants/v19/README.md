# Варіант 19 — Повний приклад виконаної лабораторної

Це **повністю розв'язаний варіант 19** лабораторної роботи №2. Використовуйте як зразок разом з `demo/`.

## Різниця між demo/ та v19/

|          | demo/                                  | v19/                              |
| -------- | -------------------------------------- | --------------------------------- |
| Що це    | Інтерактивний приклад від викладача    | Повне рішення варіанту 19         |
| Дані     | Демонстраційні (не з жодного варіанту) | Точно з v19.md                    |
| Для чого | Зрозуміти як працює кожне завдання     | Побачити як виглядає здана робота |

## Особливості варіанту 19

| Завдання     | Що робити                                                           |
| ------------ | ------------------------------------------------------------------- |
| 1 Replace    | Заміна "ро" на "ри" в тексті про професора                          |
| 2 Sort       | Сортування міст за довжиною назви, потім за алфавітом               |
| 4 Dates      | Дні тижня українською (неділя — четвер)                             |
| 5 Password   | Пароль без підрядка логіну (≥3 символи), довжина 16                 |
| 6 Duplicates | Вибір унікальних елементів з масиву                                 |
| 8 Arrays     | Обʼєднання масивів → видалення дублікатів → сортування за спаданням |
| 9 Assoc      | Студенти та їх оцінки (1-12), сортування за іменем і оцінкою        |
| 10 Form      | Логін: "valeriya_b19", 10 міст, вибір мови                          |
| 11 Calc      | x=5, y=4 → sin, cos, tg, степінь, факторіал                         |

## Запуск

```bash
php -S localhost:8000
```

Відкрийте: `http://localhost:8000/lr2/variants/v19/`

## Як використовувати як шаблон

1. Скопіюйте цю папку: `cp -r v30 vN` (де N — ваш номер варіанту)
2. Відкрийте ваш `vN.md` — там ваші конкретні дані
3. Замініть дані в кожному файлі на свої з `vN.md`
4. Перевірте: `http://localhost:8000/lr2/variants/vN/`

> **Не змінюйте цю папку** — вона доступна всім студентам як приклад.
